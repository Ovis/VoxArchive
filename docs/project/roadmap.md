# ロードマップ

## マイルストーン

## M1: 契約固定と基盤整備
- 目的: Domain/Application/Audio/Encoding の契約を固定し、ビルド可能な土台を作る。
- 完了条件:
  - ソリューションと各 Abstractions プロジェクトがビルド成功
  - 主要インターフェースが仕様と整合

## M2: 録音最短E2E
- 目的: Speaker + Mic をFLACに保存する最短パスを通す。
- 完了条件:
  - Start/Stop でFLAC生成
  - CH1=Speaker, CH2=Mic が成立

## M3: 同期安定化
- 目的: ドリフト補正（PI+可変レート）を導入し長時間安定化。
- 完了条件:
  - 1時間録音で実用レベルの同期維持
  - underflow/overflow 指標が取得可能

## M4: UI統合
- 目的: WPF + MVVM で操作/UI状態反映を完成。
- 完了条件:
  - 状態遷移・ボタン活性制御・レベル表示が仕様通り

## M5: Process Loopback + Failover
- 目的: アプリ指定録音と一方向フォールバックを実装。
- 完了条件:
  - 起動時チェック
  - 録音中終了検知で Process -> Speaker 切替
  - 自動復帰なし

## M6: 試験・品質固め
- 目的: 単体・結合・長時間試験の定着。
- 完了条件:
  - 主要ユースケースの回帰検証可能
