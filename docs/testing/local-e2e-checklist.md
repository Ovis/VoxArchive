# ローカル E2E チェックリスト

最終更新: 2026-03-17

## 1. 起動
- [ ] アプリが起動する
- [ ] メイン画面が表示される
- [ ] トレイアイコンが表示される

## 2. 録音基本
- [ ] 録音開始 -> 一時停止 -> 再開 -> 停止が動作
- [ ] 出力 FLAC が作成される
- [ ] 停止後にライブラリへ反映される

## 3. 録音モード
- [ ] スピーカーモードで音声取得できる
- [ ] プログラムモードで対象プロセスの音声取得できる
- [ ] 対象喪失時にスピーカーモードへフォールバックする

## 4. ライブラリ
- [ ] 再生/停止/シーク/倍速が動作
- [ ] タイトル保存/リネームが動作
- [ ] 一覧削除/ファイル削除が動作
- [ ] モノラル保存（WAV/MP3/FLAC）が動作

## 5. 文字起こし
- [ ] 手動文字起こしが実行できる
- [ ] txt/srt/vtt/json が出力される
- [ ] 文字起こし結果を開くが動作
- [ ] 自動文字起こし設定時、録音停止後にキュー投入される

## 6. 設定
- [ ] ショートカット変更が反映される
- [ ] 保存先変更が反映される
- [ ] 環境チェックが表示される
